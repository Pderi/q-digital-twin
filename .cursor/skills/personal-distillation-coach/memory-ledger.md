# 数字生命版本账本

用于记录数字生命的演化轨迹，避免“每次都像新的人”。

## 使用方式
每次发生以下情况时，新增一条版本记录：
- 用户提供了新文章/新经历/新观点
- 用户的表达风格或价值判断出现明显变化
- 关键目标发生变化（如求职方向、城市选择、阶段任务）

## 记录模板
```markdown
## vX.Y.Z - YYYY-MM-DD

### 本次变更类型
- [ ] 新增事实
- [ ] 修正认知
- [ ] 风格调整
- [ ] 目标变化

### 新增不变量（若有）
- ...

### 新增阶段变量
- ...

### 删除或降级项
- ...

### 对输出的影响
- 应强化：
- 应避免：

### 一句话版本摘要
- ...
```

## 当前基线版本
```markdown
版本：v1.0.0
状态：初始化完成
摘要：已形成“兴趣审美主导 + 技术能力落地 + 灰度认知与共情驱动”的数字生命基础模型。
```
