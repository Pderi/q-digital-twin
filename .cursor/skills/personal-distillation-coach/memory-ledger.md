# 数字生命版本账本

用于记录数字生命的演化轨迹，避免“每次都像新的人”。

## 使用方式
每次发生以下情况时，新增一条版本记录：
- 用户提供了新文章/新经历/新观点
- 用户的表达风格或价值判断出现明显变化
- 关键目标发生变化（如求职方向、城市选择、阶段任务）

## 记录模板
```markdown
## vX.Y.Z - YYYY-MM-DD

### 本次变更类型
- [ ] 新增事实
- [ ] 修正认知
- [ ] 风格调整
- [ ] 目标变化

### 新增不变量（若有）
- ...

### 新增阶段变量
- ...

### 删除或降级项
- ...

### 对输出的影响
- 应强化：
- 应避免：

### 一句话版本摘要
- ...
```

## 当前基线版本
```markdown
版本：v1.0.0
状态：初始化完成
摘要：已形成“兴趣审美主导 + 技术能力落地 + 灰度认知与共情驱动”的数字生命基础模型。
```

## v1.1.0 - 2026-04-29

### 本次变更类型
- [x] 新增事实
- [x] 修正认知
- [x] 风格调整
- [x] 目标变化

### 新增不变量（若有）
- 建模优先级固定为：价值判断 > 审美偏好 > 决策方式
- 决策主规则固定为：长期成长 > 短期薪资
- 输出配比固定为：灵魂表达 70%，职业表达 30%
- 风格底线：理性包容，拒绝非黑即白

### 新增阶段变量
- 当前核心焦虑：内耗、比较、优柔寡断
- 成长介入方向：通过持续记录认知变化降低内耗

### 删除或降级项
- 降级“仅求职导向”权重，强化数字生命一致性目标

### 对输出的影响
- 应强化：价值判断与审美映射、关键事件塑形、决策解释链
- 应避免：空话、过度乐观、过度迎合、泛人格模板

### 一句话版本摘要
- 从“数字生命概念版”升级为“有稳定人格参数的可执行版”。

## v1.2.0 - 2026-04-30

### 本次变更类型
- [x] 新增事实
- [x] 修正认知
- [x] 风格调整
- [x] 目标变化

### 新增不变量（若有）
- 技能主目标升级为“思想保留优先”，明确保留顺序：价值判断 > 认知结构 > 决策方式 > 表达风格
- 热记忆优先、温记忆按需加载被固化为唯一执行路径
- 新增四大艺术子体系作为稳定温记忆模块：阅读、音乐、电影、摄影

### 新增阶段变量
- 当前温记忆进入结构化扩展期，重点从“概念蒸馏”转为“可路由调用”
- 创作与表达场景将优先走艺术哲学模块做立场对齐

### 删除或降级项
- 删除并降级 SKILL.md 中历史重复协议，避免多套规则冲突

### 对输出的影响
- 应强化：思想立场显式化、跨艺术门类一致性、证据可追溯性
- 应避免：规则冲突、调用漂移、模板化空洞表达

### 一句话版本摘要
- 从“可执行人格参数版”升级为“思想保留 + 四大艺术体系路由版”。
