# references 使用说明

这个目录用于存放 `personal-distillation-coach` 技能的温记忆文档素材。

## 与热记忆的关系
- 热记忆：`../memory.md`（优先读取）
- 温记忆索引：`WARM_MEMORY_INDEX.md`（按需选文）
- 本目录文档：仅在热记忆信息不足时按主题读取

## 推荐分类

- `music/`：音乐审美、专辑分析、歌词理解、听感记录
- `film/`：电影/剧集理解、导演风格、观影笔记
- `photo/`：摄影哲学、拍摄审美、器材与后期原则
- `books/`：阅读笔记、思想摘录、哲学/心理学输入
- `life/`：生活感悟、幸福观、认知变化记录
- `career/`：实习复盘、求职准备、职业规划材料
- `profile/`：个人画像、价值观、长期目标等稳定资料

## 命名规范（建议）

- 文件名格式：`YYYY-MM-DD-主题关键词.md`
- 示例：
  - `2026-04-29-音乐审美阶段总结.md`
  - `2026-04-29-电影观转变记录.md`
  - `2026-04-29-秋招自我介绍迭代.md`

## 写作建议（便于 Skill 高质量引用）

每篇文档尽量包含以下模块：

1. 背景与场景
2. 核心观点（3-5 条）
3. 具体例子或作品证据
4. 可迁移的方法（如何复用到表达/求职/创作）
5. 一句话结论

## 使用建议

- 优先把“兴趣爱好 + 世界观 + 关键经历”的材料放全。
- 新增文档后，先更新 `WARM_MEMORY_INDEX.md` 再供 Skill 按需调用。
- 重要资料建议同步沉淀到 `reference.md` 的稳定画像部分。
