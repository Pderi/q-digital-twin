# 结束第一份实习后的胡思乱想

在2025-12-12，我结束了我的第一份实习，实习期间我又重新回到了高中作息，每天早上被六点的闹钟叫醒的时候，每天高强度脑力工作五六个小时后回到学校还要面临一大堆实验和作业的时候，每次加班到晚上十点的时候，我都想立刻提前结束实习，但最终还是坚持下来了。可终于到了离职这一天时，我却还是有些不舍和伤感，人类好像总是这样矛盾 

时长三个月的实习，大概可分为三个阶段

第一阶段:前一个半月，初到公司，接触到学校外的世界，一切都很新鲜，每天在公司一天时间学到的东西比在学校一周学到的东西都多，这个阶段每天上班都充满动力，下班后也不会感觉到疲倦，有的只是一种收获知识的满足感

第二阶段: 一个半月过后，我逐渐进入了疲软期，随着新鲜感褪去，每天工作内容慢慢开始有些重复，自己好像也变得越来越麻木，对日常生活的感知力好像也越来越差

第三阶段:实习结束前一周，我开始意识到。第二阶段才是人生中的常态，第一阶段并不多见。但是如何面对第二阶段中的这种"日复一日"把人区分成了不同的样子，也就是著名的ABC理论，影响你心态的并不是事情本身，而是你对这件事情的看法，你的情绪是由你的思维主导的。同样是工作、是磨难。有人收获的是阅历、有人收获的是年龄。有人得到的是苦难、有人得到的是成长。同时同境，心境决定一切

这篇文章中我想要重点谈论的，也正是我的第二阶段和第三阶段，因为相比于第一阶段的高光，第二阶段和第三阶段的平淡和重复才是人生的常态。就像西西弗斯一样，“疲软期、重复、麻木、感知力下降”，这几乎是现代职场版“西西弗斯推石上山”的精准写照。古希腊神话中，西西弗斯受罚永无止境地将巨石推上山顶，而后石头滚落，他再下山重复这一过程。这不正是日复一日的重复工作带来的精神消耗吗？那种新鲜感褪去后，纯粹因责任而驱动的、看似无意义的循环，会让人产生深刻的倦怠与怀疑。 但加缪在《西西弗斯的神话》中，提出了一个震撼的命题：**“我们必须想象西西弗斯是幸福的。”** 他的幸福与力量，不源于改变命运（石头注定会滚落），而源于**清醒的认知和内心的选择**。当他清醒地认识到命运的荒谬，并以蔑视和全情投入的姿态，去完成每一次推石上山的动作时，他就超越了自身的命运。他主宰了这段过程的意义。而能否让我们从“受罚的西西弗斯”，一跃成为**加缪笔下的“幸福西西弗斯”，这是我认为打工人最重要的转变之一。而又要如何达成这一转变呢？**

**我想到了一部电影《时空恋旅人》，电影中有一个设定，男主角蒂姆家族的男人都拥有穿越时空的能力，这看似是解决一切遗憾的终极利器，也大概率是一部爽片吧？但这部电影之所以成为经典，正是因为他的后半部分，在男主无数次利用穿越时空的超能力去实现一个完美的人生后，仍然感觉不满足，最终在电影最高潮，男主的父亲告诉他使用穿越时空这份超能力的终极秘密：幸福生活的秘诀，是过两遍每一天。第一遍，在慌乱和压力中经历它；第二遍，用同样的经历，但用心去感受其中被忽略的美好细节。而我们完成上述说的向加缪笔下的“幸福西西弗斯”的转变的秘诀，也正是这句话。它最终告诉我们：真正的幸福和成长，并不来自修改过去或追求永恒的新奇，而恰恰来自学会如何去拥抱和热爱那“日复一日”的、不可修改的生活本身。**

**看似日复一日的上班生活，其实藏了很多值得我们去发现的幸福，比如上下班路上遇到的形形色色的人，我们可以去观察每个人的表情与动作，去想象每个人背后都藏着什么故事；我们可以观察上下班路上的落叶，去感受季节的更迭；我们可以观察早上上班时太阳光洒下来，路面反射太阳光，也可以观察下班时的月光；今天上班可以去走一条新的上班路线；今天上班可以在耳机里听一些自己之前没接触过的类型的音乐等等。日常生活中有太多小美好和小幸福了，它是客观存在的，它就在那里，等着你去发现。而当你培养出这一双眼睛后，你会发现，整个世界焕然一新。这一生志愿只要平凡快乐，谁说这样不伟大呢？一个人能观察落叶，鲜花，从细微处欣赏一切，生活就不能把他怎么样。生活不是只有功成名就后的金碧辉煌，还有当下每一个能用心感受的瞬间。**
