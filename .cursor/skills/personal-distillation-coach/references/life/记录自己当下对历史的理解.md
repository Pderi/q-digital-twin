# 记录自己当下对历史的理解

（注:此篇文章更多的目的是记录当下自己对历史的理解，带有强烈主观色彩）

自24年末观看了电影《让子弹飞》后，便打开了我探索历史的大门，随着这两年接触到的信息越来越多，感觉已经到达了一个临界点，便写篇文章将20岁的自己对历史的理解记录下来。

首先，我想先列出自己目前总结出的一个历史模型，它并不严谨，也未必正确，但真实地反映了我此刻对历史运行方式的直观判断：:

天然状态-->偶然因素导致阶层分化-->上位的既得利益者制定规则维护自己的阶层-->上层持续压迫底层-->底层互相伤害-->到达临界点-->底层中一部分团结起来爆发革命-->重新洗牌-->新上位的成为新的恶龙-->不断循环。

循环的终点或许是人类造出硅基生物，完成了人类这个碳基生物的历史使命，之后便没必要存在了，人类的作用只是硅基生物的开机键，成功完成碳基生物向硅基生物的转换，但是，硅基生命的出现可能不是循环的“结束”，而是循环演变成了一个我们尚无法理解的、更宏大的新篇章。而人类的价值，或许就在于我们既是这个循环的“造物”，也是第一个试图理解并亲手绘制下一个篇章的“作者”

这篇文章，我想以我总结出的这个历史模型来展开。不难看出，对于历史我的理解非常悲观，历史从无正义，无止境的循环，无止境的剥削与压迫，无止境的阶层固化，好像一切都没有尽头。好像一切都是虚无，《三体》中让叶文洁下定决心按下那个按钮的驱动力，或许也是因为她对人类的未来深感悲观。

那么问题来了，我们难道就只能向死而生吗，既然一切指向的都是虚无，都是无止境的循环，我们的生活还有意义吗，对此有段时间我也深陷困扰。人生有三种阶段:

1.见山是山，见水是水  

2.见山不是山，见水不是水  

3.见山还是山，见水还是水   

有段时间我好像一直被困在了第二阶段，感觉一切都是虚无，但随着看的书越来越多，看的电影越来越多，一直困扰我的虚无主义好像迎来了曙光。

一个人能观察落叶，鲜花，从细微处欣赏一切，生活就不能把他怎么样。答案好像就藏在这句话里。当看透了历史的残酷和系统的无情之后，不需要变得愤世嫉俗，也不需要变得虚无主义，而是选择转过身用一种“接地气”的方式，去过好自己这一生。也就是:放弃对“宏观世界"的幻想式掌控，转而追求“微观世界”的确定性幸福。

在“宏观”上，你要承认这个世界有“系统”，有“规律”，有“宿命”。你个人，大概率是改变不了什么的。但在“微观”上，在属于你自己的那个小世界里，你又是绝对自由的。你可以选择读什么书，爱什么人，做什么工作，把时间花在什么地方。

正如《明朝那些事》里最后说到:

**所谓百年功名、千秋霸业、万古流芳，与一件事情相比，其实算不了什么。**

**这件事情就是——用你喜欢的方式度过一生**

**这就是历史教给我的:放弃对“宏观世界"的幻想式掌控，转而追求“微观世界”的确定性幸福。这一生志愿只要平凡快乐，谁说这样不伟大呢？在日常生活中，我也在通过读书，看影视剧，摄影，听音乐等方式来不断强大自己的内核**

**写到这里，好像已经足够自洽，但我突然想到了之前看的《浪浪山小妖怪》的一篇影评中的一段话:**

他们从英雄梦中醒来，变回了真正的chu生，除此以外，似乎什么都没有改变。

那他们这一路的拼搏，这一路的努力，又有什么意义呢？

黄眉怪也问过小猪妖这个问题，他不理解，放着好好的日子不过，你到底想要什么？

小猪妖的回答是：

“我想要活成我喜欢的样子。”

我不知道你们听到这个回答是什么感觉，我是失望的，这其实不是小猪妖的回答，这是主创的回答。

当现实的壁垒越来越森严，当上升的渠道越来越狭窄，当虚幻的承诺越来越难兑现，当外界的成功越来越不可能，只能退回到内心秩序的建立，才能赋予努力以意义。

我上面所写到的放弃对“宏观世界"的幻想式掌控，转而追求“微观世界”的确定性幸福好像就是退回到内心秩序的建立来赋予努力意义，好像是因为没得选了我才会这样说，我好像是被动做出的选择，是这样吗，现在的我也不是很能理解，便写下此篇文章，记录我目前的困惑留给未来的自己来进行解答。我好像仍然处在见山不是山，见水不是水的人生第二阶段。又或者说，我被卡在了第二阶段和第三阶段之间

希望未来的我可以解答此刻20岁的自己的困惑
