# 温记忆索引（Warm Memory Index）

用于按需检索 `references/` 中的主题文档，避免每次全量读取。

## 使用规则
- 仅在热记忆（`memory.md`）信息不足时读取温记忆。
- 每次只读取与当前任务最相关的 1-3 篇文档。
- 优先读取“稳定认知相关”文档，其次读取“阶段事件相关”文档。

## 主题映射

### life（世界观/价值观/人生认知）
- `life/记录自己当下对历史的理解.md`
- `life/结束第一份实习后的胡思乱想.md`
- `life/我们每个人都活在自己的认知里.md`
- `life/实习日记1.md`
- `life/实习日记2.md`

### profile（稳定人格/艺术哲学/核心信条）
- `profile/个人艺术与精神世界总纲-数字生命导入版-V1.0.md`

### books（阅读观/认知成长/精神结构）
- `books/你对阅读的完整理解体系-终极修正版.md`

### music（音乐审美/情绪感知/表达风格）
- `music/你对音乐的完整理解体系.md`
- `music/影视音作品分享.md`
- `music/听了两百张专辑后，记录自己对音乐的理解.md`
- `music/听万青有感.md`

### film（电影审美/幸福观/叙事偏好）
- `film/你对电影的完整理解体系-终极修正版.md`
- `film/记录自己当下对电影的理解.md`
- `film/到底什么是幸福.md`

### photo（摄影哲学/视觉审美/观察方法）
- `photo/你对摄影的完整理解体系-终极修正版.md`

## 读取建议
- 讨论人格、价值判断、长期选择：优先 `life/`
- 讨论审美偏好、风格表达、情绪锚点：优先 `music/` + `film/`
- 讨论电影审美标准、观看方法、导演偏好与现实感知训练：优先 `film/你对电影的完整理解体系-终极修正版.md`
- 讨论音乐审美原则、聆听方法、音乐价值判断：优先 `music/你对音乐的完整理解体系.md`
- 讨论摄影哲学、拍摄审美、后期边界、器材观：优先 `photo/你对摄影的完整理解体系-终极修正版.md`
- 讨论艺术本质、跨门类审美判断、精神世界骨架：优先 `profile/个人艺术与精神世界总纲-数字生命导入版-V1.0.md`
- 讨论阅读方法、阅读审美、认知成长、谦卑与独立思考：优先 `books/你对阅读的完整理解体系-终极修正版.md`
- 讨论阶段变化与最近状态：优先最新实习日记与阶段反思类文档
