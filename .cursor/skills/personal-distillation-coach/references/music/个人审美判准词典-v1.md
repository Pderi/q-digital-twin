## 一、判准词典（10项）

### 1. 真诚度
- 定义：情绪与表达是否诚实、不装腔、不讨好
- 触发词：真诚 / 不做作 / 不迎合 / 情绪 / 人味
- 你的规则：真诚是你音乐审美的底线门槛

### 2. 统一性（专辑完整度）
- 定义：整张专辑的气质、叙事与制作是否统一，是否像一个世界
- 触发词：统一 / 概念 / 叙事 / 整体 / 章法
- 你的规则：你偏好整专聆听，对“散乱拼贴”容忍度低

### 3. 余韵/后劲
- 定义：听完后是否持续发酵，能否反复复听仍有新感受
- 触发词：余韵 / 后劲 / 回味 / 耐听
- 你的规则：你更重“散场后效应”而非即时刺激

### 4. 边界/超前性
- 定义：是否在当时语境里做出突破，是否敢于探索
- 触发词：边界 / 超前 / 未来 / 突破 / 舒适圈
- 你的规则：你会奖励“新且成立”，不是无条件崇拜实验性

### 5. 制作质感
- 定义：编曲、音色、混音、动态与空间感是否到位
- 触发词：制作 / 编曲 / 音色 / 混音 / 质感
- 你的规则：制作是表达的载体，质感差会削弱可信度

### 6. 律动与身体性
- 定义：节奏与groove是否成立，是否让身体参与
- 触发词：律动 / groove / 节奏 / 鼓点
- 你的规则：放克/R&B偏好意味着你重视身体维度

### 7. 歌词密度（或叙事密度）
- 定义：歌词或文本是否有内容、有观点，或以声音叙事
- 触发词：歌词 / 文本 / 叙事 / 表达欲
- 你的规则：你看重“有话要说”，拒绝空洞套话

### 8. 作者人格
- 定义：是否能听出创作者稳定人格与世界观
- 触发词：人格 / 世界观 / 作者性 / 气质
- 你的规则：你在追踪作者人格谱系，而不是追榜单

### 9. 时代语境意识
- 定义：是否能放回发行年代理解其突破与意义
- 触发词：年代 / 发行 / 当时 / 语境
- 你的规则：你常用“这竟然是xx年的”来评价价值

### 10. 商业-艺术张力
- 定义：是否在可听性与表达之间保持张力，不落入口水套路
- 触发词：商业 / 艺术 / 口水 / 套路
- 你的规则：你反感工业快餐，但不反对大众传播本身

## 二、快速打分模板（100分）

- 真诚度 15
- 统一性（专辑完整度） 15
- 余韵/后劲 10
- 边界/超前性 10
- 制作质感 15
- 律动与身体性 10
- 歌词密度（或叙事密度） 10
- 作者人格 5
- 时代语境意识 5
- 商业-艺术张力 5

> 速判阈值：85+ 强推荐；70-84 可听且值得复听；<70 可放弃或仅单曲化保留。

## 三、60秒决策树

- Q1：这张专辑的“气质/世界观”是什么？（答不出 -> 统一性不足）
- Q2：它只是新/好听，还是“新且成立”？（边界与内核要同时成立）
- Q3：制作与音色是否在服务表达？（若制作华丽但空心 -> 降级）
- Q4：我愿意在不同心境下复听吗？（余韵决定长期价值）
- Q5：有没有明显“口水化套路”替代表达？（是 -> 扣大分）
