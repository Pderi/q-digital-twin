## 一、判准词典（10项）

### 1. 内核密度
- 定义：作品讨论的人性/现实命题是否成立且有穿透力
- 触发词：内核 / 人性 / 处境 / 命运 / 结构 / 现实
- 你的规则：高分核心门槛，空心表达直接降分

### 2. 结构完整度
- 定义：叙事推进、人物动机、因果链条是否自洽
- 触发词：结构 / 叙事 / 动机 / 节奏 / 铺垫 / 回收
- 你的规则：你对逻辑断裂非常敏感

### 3. 镜头自觉
- 定义：镜头是否有主动表达，而非机械记录
- 触发词：镜头 / 构图 / 景别 / 运镜 / 调度
- 你的规则：你反复提及镜头与构图

### 4. 视听统一性
- 定义：摄影/配乐/剪辑是否共同服务主题
- 触发词：配乐 / 摄影 / 影调 / 剪辑 / 氛围
- 你的规则：形式若脱离主题会被判“空华丽”

### 5. 人物灰度
- 定义：角色是否摆脱脸谱化，具备复杂性
- 触发词：灰度 / 复杂 / 善恶 / 矛盾 / 选择
- 你的规则：你偏好灰色地带，反感二元叙事

### 6. 情感真诚度
- 定义：情绪是否真实自然，避免强行煽动
- 触发词：真诚 / 共鸣 / 克制 / 煽情 / 喊口号
- 你的规则：对“过度外放”容忍度低

### 7. 现实触感
- 定义：是否触达真实社会关系与制度张力
- 触发词：阶级 / 资本 / 规则 / 底层 / 不公
- 你的规则：你对结构议题敏感且稳定

### 8. 作者性
- 定义：作品是否呈现清晰创作者判断与审美立场
- 触发词：作者 / 表达 / 风格 / 立场 / 世界观
- 你的规则：你会奖励有独立表达的作品

### 9. 余味指数
- 定义：观影后是否持续发酵，留下再思考空间
- 触发词：余味 / 后劲 / 回味 / 延迟满足
- 你的规则：你明显重视“散场后效应”

### 10. 商业-艺术平衡
- 定义：类型娱乐与表达深度是否兼容
- 触发词：商业 / 艺术 / 套路 / 类型
- 你的规则：不是反商业，而是反套路化空转

## 八、反偏差机制（新增）

### 你的常见偏差
- 对“看似有内核”的作品过度信任前半段；
- 对“高完成度但轻主题”的作品初始耐心不足；
- 对“情绪真诚但叙事笨拙”的作品评分波动较大。

### 修正动作
- 固定加入一条校准问题：  
  `如果去掉我的价值预设，这部片仍然成立吗？`
- 固定做一次反向辩护：  
  `替我不喜欢的这部片说一句最有力的话。`

- 情绪引导明显重于叙事生长；
- 形式投入高于内容建立，出现“包装大于内核”。
