# 记录自己当下对电影的理解

注：艺术感受一般或多或少是主观的，这篇文章也不可避免的展现出很强的主观性，如有不适，还请谅解
开头，我想先给出目前我个人看电影经历的五个阶段：
第一阶段：睁开眼睛，最开始看中国电影，认识到电影是什么
第二阶段：俯视。慢慢开始看外国电影，这时候会感觉打开新大陆，觉得外国电影太强了，完全领先于国内，这确实是客观事实，因为1978年中国才改革开放，整体环境外国确实比中国领先几十年。但我会出现一种很奇怪的觉得自己的审美更高级的傲娇感
第三阶段：平视，随着阅片量上升会觉得客观来讲中国很多电影确实在镜头方面，特效方面等等不如外国一些电影，但在中国的土壤下，也确实有比如《让子弹飞》《大佛普拉斯》这种本土叙事在外国找不到替代品的作品，承认外国电影的先进，但同时不通过对比来贬低中国电影，并且意识到中国电影也有一些不可替代的地方
第四阶段：感受与共鸣，这是我目前的状态，镜头，构图，运镜等技术运用我认为只是引起观众共鸣的工具。相比之下主观感受与共鸣在我的评分体系中占比会稍偏大，也就是电影的内核和灵魂，有时甚至会占绝对主导位置，这是我目前的局限。
而第五阶段是我目前没能达成的：既能在技术层面冷静拆解，又能在感受层面彻底投入，在个人评分体系中使主观感受与客观分析达成一个和谐的平衡，就像那些优秀艺术家在商业效果和艺术表达达成平衡一样，但我必须承认，我还没有达成这个阶段，这是我目前努力的方向
我想重点聊聊第三阶段向第四阶段的转换过程，在大学某一堂课中，老师向我们提出了一个问题：哪吒的“我命由我不由天”与李佳琦直播中说的“这么多年一直都是这个价格，有没有可能是你自己不够努力”两者之间有没有区别，有没有可能两者都是将结构性问题转化为个人奋斗问题。
顺着这个问题我去重新去审视我看过的一些电影，好像确实发现了一点不对劲的地方，客观来讲《哪吒》以及我刚刚看过的《飞驰人生3》等等类似电影，在叙事，技术层面，情绪引导方面等等确实做的非常优秀，在电影院看完至少不会觉得白来一趟，大部分人都会觉得值回票价。但在观看过后去探寻他们的内核时，会发现好像立不住，而且还很奇怪：自己就是资本却要拍电影告诉个人可以战胜资本。自己就是天，就是制定规则的人，却要拍一部电影说“我命由我不由天”，当“天”亲自下场，讲述一个“反抗天”的故事时，荒诞感就诞生了。
看到过一段话：
作为规则的制定者，他们很清楚一个秘密：允许甚至鼓励对“天命”的反抗，本身就是维持“天命”最有效的方式。
电影里的“反抗”是最高潮、最迷人的戏码，但它被严格限定在荧幕之内。当灯光亮起，散场回家，现实世界的规则纹丝未动。

相比之下，那种将现实的残酷平静展现出来的作品或许更能在内核上立得住，比如《大佛普拉斯》《猜火车》《何以为家》《隐入尘烟》等，他们不提供解决方案，他们撕开了幻想，展示了现实，他们没有营造假惺惺的善良，他们让我们直视不公，直视不适。但这些电影又有一个局限性，那就是他们或许并不会吸引到很多人来到电影院观看，这就又来到了一个经典问题：艺术表达与商业效果的平衡。 《让子弹飞》等电影口碑票房双丰收就是这个逻辑，因为它几乎完美地达到了平衡，做到了雅俗共赏。

或许正因如此，类似姜文，方大同等等这样的艺术家们才会如此具有人格魅力，很多人在评价姜文电影时也会带着一层滤镜，正如那句评价《你行！你上！》的“六分的电影，十分的姜文”，因为他们能在艺术表达和商业效果之间能保持一种微妙的平衡，并构建出一种鲜明的个人特色，他们属于“站着把钱挣了”。
这便是我由第三阶段到达第四阶段的过程的一个缩影。

电影从来都是时代的镜像，希望每个人都可以在现代社会找到自己与电影共存的方式，

回看开头提出的五个观影阶段，我突然意识到，这何尝不是我们认识世界的过程？年少时非黑即白（俯视/仰视），长大后开始理解灰色地带（平视），再到后来，我们被那些触及灵魂的东西打动，也学会了警惕那些过度的煽动。
而那个我尚未达到的‘第五阶段’，或许并不仅仅是看电影的境界，它更像是一种理想的人生姿态：面对这个世界，既要有拆解其运行逻辑的冷峻，又要有投入去爱、去感受的热忱。正如《开罗紫玫瑰》表达的，电影散场了，生活还在继续。愿我们都能在属于自己的那部人生电影里，早一点学会，站着，把这条路走完。

共勉
